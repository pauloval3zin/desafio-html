Titulos: h1 até h6

As tags h1 a h6 são usadas para definir cabeçalhos em uma página HTML. Cada tag representa um nível de cabeçalho diferente, com h1 sendo o nível mais alto e h6 sendo o nível mais baixo.

Os cabeçalhos são importantes para a estrutura e a organização de uma página HTML. Eles ajudam os usuários a navegar pela página e a entender o conteúdo. Os cabeçalhos também são usados por mecanismos de pesquisa para indexar e classificar páginas da web.

É importante usar os cabeçalhos de forma consistente. O cabeçalho h1 deve ser usado apenas uma vez na página e deve representar o assunto geral da página. Os cabeçalhos h2 a h6 devem ser usados para subtítulos e para dividir o conteúdo da página em seções menores.

Os cabeçalhos também podem ser usados para estilizar o texto na página. Por padrão, os cabeçalhos são exibidos em um tamanho de fonte maior e em um estilo de fonte diferente do texto do corpo. No entanto, os estilos dos cabeçalhos podem ser personalizados usando CSS.

Tag: < p >

A tag p é uma tag HTML que representa um parágrafo de texto. É um elemento de nível de bloco, o que significa que ele ocupa a largura total de seu contêiner pai. Por padrão, os navegadores adicionam automaticamente uma única linha em branco antes e depois de cada elemento < p >.

A tag p pode ser usada para agrupar texto relacionado, como uma lista de itens ou um bloco de texto que é recuado do resto da página. Também pode ser usado para adicionar espaço entre parágrafos ou para criar uma nova linha sem adicionar nenhum conteúdo.

Tag: mark

A tag < mark > é uma tag HTML que representa um trecho de texto que é marcado ou destacado para referência ou anotação. O texto marcado pode ser um termo, uma frase ou uma palavra. É usado para chamar a atenção do usuário para uma parte importante do texto.

A tag < mark > é um elemento de nível de bloco, o que significa que ele ocupa a largura total de seu contêiner pai. No entanto, o texto marcado é normalmente estilizado para se destacar do texto circundante, como com uma cor de fundo diferente ou uma fonte diferente.

Tag: small

A tag < small > é uma tag HTML que representa um texto menor do que o texto padrão da página. O texto marcado com a tag < small > é geralmente usado para texto legal, avisos de direitos autorais ou outras informações que são importantes, mas não precisam estar em destaque.

A tag < small > é um elemento de nível de bloco, o que significa que ocupa a largura total de seu contêiner pai. No entanto, o texto marcado com a tag < small > é normalmente estilizado para ser menor do que o texto padrão da página, como com um tamanho de fonte menor ou uma cor de fonte diferente.

    Tag: hr

    A tag hr é um elemento HTML que representa uma linha horizontal. É usado para separar diferentes seções de conteúdo em uma página da web. A tag hr não tem conteúdo, mas pode ser estilizada usando CSS.

    A tag hr é um elemento de nível de bloco, o que significa que ocupa a largura total de seu contêiner pai. A tag hr é normalmente exibida como uma linha horizontal, mas pode ser estilizada para ter uma aparência diferente usando CSS.

    Tag: sub

    O elemento HTML sub é usado para representar texto subscrito. O texto subscrito é texto que é exibido abaixo da linha de base do texto normal. O elemento sub é normalmente usado para exibir fórmulas matemáticas ou outras informações que precisam ser exibidas abaixo do texto normal.

    O elemento sub é um elemento de nível de linha, o que significa que ele ocupa apenas o espaço necessário para o seu conteúdo. O elemento sub não tem conteúdo, mas pode ser estilizado usando CSS.

    Tag: sup

    O elemento HTML sup é usado para representar texto sobrescrito. O texto sobrescrito é texto que é exibido acima da linha de base do texto normal. O elemento sup é normalmente usado para exibir fórmulas matemáticas ou outras informações que precisam ser exibidas acima do texto normal.

    O elemento sup é um elemento de nível de linha, o que significa que ocupa apenas o espaço necessário para o seu conteúdo. O elemento sup não tem conteúdo, mas pode ser estilizado usando CSS.
